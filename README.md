# online resume for students

## Prerequisites

- [GitHub account]
- [Azure account]
- [Azure CLI]
- [.NET Core 3.1 LTS]
- [Azure Functions Core Tools]
- [Visual Studio Code]
- [Visual Code Extensions]
  - [Azure Functions Extensions]
  - [C# Extension]
  - [Azure Storage Extension]
